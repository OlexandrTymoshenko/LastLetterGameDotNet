Last letter game developed using .Net Core 2.1 SDK 

#How to run:

1. Install .Net Core 2.1 SDK https://dotnet.microsoft.com/download/dotnet-core/2.1
2. Open cmd in LastLetterGameDotNet folder
3. dotnet restore
4. dotnet run
